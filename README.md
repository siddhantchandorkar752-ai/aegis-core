# Aegis Core (NEURO-CORE)

An interactive Streamlit dashboard for real-time neural execution and adversarial auditing.

## Architecture Features
- **Real-Time Research Simulator**: Dynamic tracking of compute-to-market metrics.
- **Neural Execution Engine**: Train linear predictors via PyTorch on synthesized data.
- **Adversarial Auditor**: Injection vulnerability scanning and automated cryptographic validation using Merkle trees.

## Running Locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the Streamlit server:
   ```bash
   streamlit run main.py
   ```
